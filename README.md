'# vibe-product-runbook

## 安装这个 Skill（推荐先读）

### 方法一（直接安装仓库）
```bash
npx skills add https://github.com/Ogurijay/vibe-product-runbook --skill vibe-product-runbook
```

### 方法二（你也可以先 clone 后本地引用）

```bash
# 如果你已经有本地仓库
npx skills add <本地路径或远端仓库地址> --skill vibe-product-runbook
```

> 安装完成后，按项目要求运行会创建/读取项目级 `.vibe/*` 文件用于会话续航。


# 内容说明
- 这是一个项目级 AI 开发会话记忆模板 skill，用于启动/继续项目与自动沉淀复用经验。
